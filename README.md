# Mohammad Abu-Khader - Portfolio Site

Personal portfolio deployed via GitHub Pages at [xoity.github.io](https://xoity.github.io).

## Stack

Friend-clone static portfolio: HTML, inline CSS, vanilla JS, Three.js, GSAP, GitHub API project sync, JSON-driven certifications, downloadable CV, and mailto contact workflow.

## Local Preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
